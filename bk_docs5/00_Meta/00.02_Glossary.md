# 社内用語集・略語集

> **テンプレート利用ガイド**
> 全職種が共通理解を持つための用語集です。
> プロダクト固有の用語、業界用語、社内略語を追加してください。
> アルファベット順・五十音順で整理すると検索しやすくなります。

## ビジネス・組織用語
| 用語/略語 | 正式名称 (Full Name) | 定義・解説 | 備考 |
| --- | --- | --- | --- |
| ARR | Annual Recurring Revenue | 年間経常収益。サブスクリプションビジネスの主要指標。 | |
| Churn Rate | - | 解約率。一定期間内に解約した顧客の割合。 | |
| FAB | Feature-Advantage-Benefit | 機能・利点・便益。製品の価値を段階的に説明する手法。 | |
| GTM | Go-To-Market | 市場投入戦略。製品を顧客に届けるための計画。 | |
| KPI | Key Performance Indicator | 重要業績評価指標。目標達成度を測る指標。 | |
| MoSCoW | Must/Should/Could/Won't Have | 要件の優先順位付け手法。 | PRDで使用 |
| MRR | Monthly Recurring Revenue | 月間経常収益。サブスクリプション収益の月次指標。 | |
| NPS | Net Promoter Score | 顧客推奨度。顧客ロイヤルティの指標（-100〜+100）。 | |
| OKR | Objectives and Key Results | 目標と主要な成果。四半期ごとの目標設定フレームワーク。 | |
| PMF | Product-Market Fit | プロダクトが市場に受け入れられている状態。 | |
| PRD | Product Requirement Document | 要件定義書。プロダクトが満たすべき仕様を記述した文書。 | |
| SMART | Specific/Measurable/Achievable/Relevant/Time-bound | 目標設定の5つの基準。 | OKRで使用 |
| TAM/SAM/SOM | Total/Serviceable/Obtainable Market | 市場規模を示す指標群。 | 市場分析で使用 |
| USP | Unique Selling Proposition | 独自の売り、強み。競合他社にはない独自の価値提案。 | |

## 技術用語
| 用語/略語 | 正式名称 (Full Name) | 定義・解説 | 備考 |
| --- | --- | --- | --- |
| ADR | Architecture Decision Record | アーキテクチャ決定記録。技術的な意思決定とその背景を残す。 | |
| API | Application Programming Interface | システム間でデータをやり取りするための仕様。 | |
| CI/CD | Continuous Integration / Continuous Delivery | 継続的インテグレーション/デリバリー。自動ビルド・テスト・デプロイ。 | |
| CRUD | Create/Read/Update/Delete | データベース操作の4つの基本機能。 | |
| E2E | End-to-End | エンドツーエンド。システム全体を通した統合テスト。 | |
| JWT | JSON Web Token | 認証情報を含むトークン形式。API認証で使用。 | |
| MQTT | Message Queuing Telemetry Transport | 軽量なメッセージングプロトコル。IoTで使用。 | |
| OAuth | Open Authorization | 認可の標準プロトコル。サードパーティ連携で使用。 | |
| REST | Representational State Transfer | Web APIの設計スタイル。HTTPメソッドを使用。 | |
| SAML | Security Assertion Markup Language | 認証情報を交換するためのXMLベースの標準規格。 | エンタープライズSSOで使用 |
| SLA | Service Level Agreement | サービスレベル契約。稼働率などの保証。 | |
| SSO | Single Sign-On | シングルサインオン。1つのIDで複数システムにログイン可能にする仕組み。 | |
| WCAG | Web Content Accessibility Guidelines | Webアクセシビリティのガイドライン。レベルA/AA/AAA。 | デザインで準拠 |

## 顧客向け用語
| 用語 | 定義 | 社内用語との対応 |
