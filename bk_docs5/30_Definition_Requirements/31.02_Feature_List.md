## 4. カテゴリー別詳細

### Writer Features (作家向け機能)

#### W-002-3: 高機能エディタ
- **概要**: ブラウザ上でストレスなく執筆できるエディタ。
- **機能**:
  - 縦書き/横書きリアルタイムプレビュー
  - ルビ記法（《》）の入力支援
  - クラウド自動保存
- **優先度**: P0 (Phase 1)
- **関連ドキュメント**: [41.01_Design_Handoff_Editor.md](../40_Design_UX/41.01_Design_Handoff_Editor.md)

#### W-002-4: 用語集管理
- **概要**: 固有名詞や独自設定の翻訳揺れを防ぐための機能。
- **ユーザーストーリー**: 作家として、主人公の名前「アルト」が「Alto」や「Art」とブレないように固定したい。
- **優先度**: P1 (Phase 2)

### Translation Features (自動翻訳)

#### T-001-1: AI自動翻訳
- **概要**: 投稿されたテキストを非同期で翻訳し、DBに保存する。
- **技術**: Claude 3.5 Sonnet / GPT-4o mini 等のLLM。
- **パフォーマンス**: 1章(3000文字)あたり30秒以内。

## 5. リリース計画

### Phase 1 (MVP)
- **Writer**: 登録、作品管理、エディタ
- **Reader**: 多言語ビューア、検索、本棚
- **Translation**: 日英翻訳、ルビ処理

### Phase 2 (Open Beta)
- **Writer**: 用語集、予約投稿、ダッシュボード(簡易)
- **Reader**: ランキング、コメント、フォロー
- **Translation**: 用語集適用、韓国語・繁体字対応

### Phase 3 (Commercial)
- **Writer**: 収益管理、お知らせ
- **Reader**: 投げ銭、オフライン、アプリ版
- **Translation**: 簡体字、欧州言語対応
