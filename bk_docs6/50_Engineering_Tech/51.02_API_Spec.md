# API仕様書 (v1)

> **テンプレート利用ガイド**
> 外部向け・内部向けAPIの仕様を定義します。OpenAPI (Swagger) 形式での管理を推奨します。
> **利用職種**: Engineer, PdM


## 概要
- **Base URL**: `https://api.star-novel.com/v1`
- **Content-Type**: application/json
- **Authentication**: Bearer Token (JWT)


## エンドポイント一覧

### 認証 (Auth)
| Method | Path | Description |
| --- | --- | --- |
| POST | /v1/auth/register | 新規ユーザー登録 |
| POST | /v1/auth/login | ログイン |

### ユーザー (Users)
| Method | Path | Description |
| --- | --- | --- |
| GET | /v1/users/me | 現在のユーザー情報取得 |
| GET | /v1/users/:userId | 公開ユーザー情報取得 |
| POST | /v1/users/:userId/follow | ユーザーをフォロー |
| GET | /v1/users/me/bookmarks | ブックマーク一覧取得 |

### 小説 (Novels)
| Method | Path | Description |
| --- | --- | --- |
| GET | /v1/novels | 小説一覧取得 (Search/Filter) |
| POST | /v1/novels | 小説作成 |
| GET | /v1/novels/:novelId | 小説詳細取得 |
| POST | /v1/novels/:novelId/bookmark | ブックマーク追加 |

### 章 (Chapters)
| Method | Path | Description |
| --- | --- | --- |
| GET | /v1/novels/:novelId/chapters/:chapterId | 章本文取得 (lang指定可) |
| POST | /v1/novels/:novelId/chapters | 章作成 |
| POST | /v1/novels/:novelId/chapters/:chapterId/translate | 手動翻訳リクエスト |

### 用語集 (Glossary)
| Method | Path | Description |
| --- | --- | --- |
| GET | /v1/novels/:novelId/glossary | 用語集一覧 |
| POST | /v1/novels/:novelId/glossary | 用語追加 |
