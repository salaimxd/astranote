> このドキュメントは、プロダクトの成功を測定するためのKPI（重要業績評価指標）を設計・追跡するためのガイドです。

## NSMとKPIの関係

```
North Star Metric（最重要指標）
  ↑
  |
Input Metrics（入力指標、KPI）
  ↑
  |
施策・アクション
```

## 重要なKPIカテゴリ

### 1. 獲得（Acquisition）

**新しい顧客をどれだけ獲得しているか**

**主要KPI:**
- **新規ユーザー数（Sign-ups）**: 新規登録者数
- **トラフィック（Traffic）**: Webサイト訪問者数
- **コンバージョン率（Conversion Rate）**: 訪問者のうち登録した割合
- **顧客獲得コスト（CAC - Customer Acquisition Cost）**: 1顧客獲得にかかるコスト

**計算式:**
```
CAC = マーケティング・営業コスト / 新規顧客数
```

**目標:**
- CAC < LTV（ライフタイムバリュー）
- CAC回収期間 < 12ヶ月（SaaSの場合）

### 2. アクティベーション（Activation）

**新規ユーザーが初期価値を体験しているか**

**主要KPI:**
- **アクティベーション率（Activation Rate）**: 初週に「Aha!モーメント」を体験した割合
- **Time to Value（TTV）**: 登録から価値体験までの時間
- **オンボーディング完了率**: チュートリアル完了率

**計算式:**
```
アクティベーション率 = Aha!モーメント達成ユーザー数 / 新規登録者数
```

### 3. リテンション（Retention）

**ユーザーが継続して使っているか**

**主要KPI:**
- **リテンション率（Retention Rate）**: N日後/N週後も使っている割合
- **チャーン率（Churn Rate）**: 解約・離脱率
- **DAU/MAU比率**: 日次アクティブ / 月次アクティブ（エンゲージメントの深さ）

**計算式:**
```
リテンション率（N日後） = N日後もアクティブなユーザー数 / 初日のユーザー数

チャーン率（月次） = 解約ユーザー数 / 月初のユーザー数

DAU/MAU比率 = 日次アクティブユーザー数 / 月次アクティブユーザー数
```

**目標:**
- リテンション率（30日後）> 40%（優秀なプロダクト）
- チャーン率 < 5%/月（SaaSの場合）
- DAU/MAU > 20%（高エンゲージメント）

### 4. 収益（Revenue）

**どれだけ収益を上げているか**

**主要KPI:**
- **MRR（Monthly Recurring Revenue）**: 月次経常収益
- **ARR（Annual Recurring Revenue）**: 年間経常収益
- **ARPU（Average Revenue Per User）**: ユーザーあたり平均収益
- **LTV（Lifetime Value）**: 顧客生涯価値

**計算式:**
```
MRR = サブスクリプション顧客数 × 平均月額料金

ARR = MRR × 12

ARPU = 総収益 / アクティブユーザー数

LTV = ARPU × 顧客生涯期間（1/チャーン率）
```

**目標:**
- LTV / CAC > 3（健全な成長）
- MRR成長率 > 10%/月（急成長スタートアップ）

### 5. 紹介（Referral）

**ユーザーが他のユーザーを呼んでいるか**

**主要KPI:**
- **バイラル係数（Viral Coefficient、K-factor）**: 1ユーザーが何人招待するか
- **NPS（Net Promoter Score）**: 推奨度
- **紹介経由の新規登録率**: 紹介プログラムからの登録割合

**計算式:**
```
バイラル係数 = 招待送信数 × 招待受諾率

NPS = 推奨者割合（9-10点） - 批判者割合（0-6点）
```

**目標:**
- バイラル係数 > 1（指数関数的成長）
- NPS > 50（優秀）、> 70（卓越）

## KPIダッシュボードの設計

### 階層構造

**レベル1: North Star Metric**
- 最重要指標を大きく表示
- トレンドグラフ（週次、月次）

**レベル2: Input Metrics（入力指標）**
- NSMに影響を与える主要KPI
- 獲得、アクティベーション、リテンション、収益

**レベル3: 詳細指標**
- チャネル別、セグメント別の内訳
- ファネル分析

### ツール

**ダッシュボードツール:**
- Google Analytics（Web解析）
- Mixpanel、Amplitude（プロダクト分析）
- Looker、Tableau、Metabase（BIツール）
- Google Data Studio（無料ダッシュボード）

**カスタムダッシュボード:**
- Grafana + PostgreSQL
- Redash
- Superset

## KPI設定のプロセス

### Step 1: NSMに影響を与える指標を特定

- NSMを分解する
- どのアクションがNSMを動かすか
- 因果関係を明確にする

### Step 2: 優先順位をつける

- 最も影響が大きい指標
- 改善可能な指標
- 5-10個に絞る

### Step 3: 目標値を設定

- 現状値を把握
- ベンチマークを調査
- 達成可能な目標を設定

### Step 4: ダッシュボード構築

- リアルタイムで追跡
- チーム全体で共有
- 定期的にレビュー

### Step 5: 継続的な改善

- 週次でKPIをレビュー
- A/Bテストで施策の効果を検証
- データに基づいて意思決定
