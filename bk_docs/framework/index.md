# ドキュメント

プロダクト開発のフレームワークへようこそ。このガイドは、アイデアの発見から拡大まで、プロダクト開発の全プロセスをサポートします。

## フレームワークの構成

### 🔍 Discovery（発見）
ユーザーのニーズを深く理解し、解決すべき課題を明確化します。

- [ペルソナ](/framework/discovery/persona) - ターゲットユーザーの具体化
- [共感マップ](/framework/discovery/empathy_map) - ユーザーの思考と感情の可視化
- [HMW](/framework/discovery/hmw) - 問題を機会に変える問いの作成

### ✅ Validation（検証）
アイデアの実現可能性とビジネスモデルを検証します。

- [リーンキャンバス](/framework/validation/lean_canvas) - ビジネスモデルの設計
- [MVP](/framework/validation/mvp) - 最小限の機能で価値を検証

### 🎯 Refinement（深化）
プロダクトの体験を詳細に設計し、開発に向けて準備します。

- [ジャーニーマップ](/framework/refinement/journey_map) - ユーザー体験の全体像
- [ユーザーストーリー](/framework/refinement/user_stories) - 機能を価値の視点で記述
- [ストーリーマッピング](/framework/refinement/story_mapping) - 機能の優先順位付け
- [テストケース](/framework/refinement/test_cases) - 品質保証の計画

### 📈 Scalability（拡大）
プロダクトの成長と組織の拡大を支える仕組みを構築します。

- [BMC](/framework/scalability/bmc) - ビジネスモデルキャンバス
- [4P分析](/framework/scalability/4p_analysis) - マーケティング戦略の設計
- [North Star Metric](/framework/scalability/northstar) - 成長の指標
- [KPI](/framework/scalability/kpi) - 重要業績評価指標
- [データ基盤](/framework/scalability/data_infra) - データドリブンな意思決定
- [ADR](/framework/scalability/adr) - アーキテクチャ決定記録
