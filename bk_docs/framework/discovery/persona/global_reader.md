## 目標・ニーズ

**達成したい目標:**

1. 東アジア（日本・中国・韓国）の小説を、原作のニュアンスを失わない高品質な翻訳で楽しみたい
2. 新しい作品を効率的に見つけ、好みに合った作品にすぐアクセスしたい
3. お気に入りの作家を直接サポートしたい（プラットフォームに搾取されない形で）

**短期的な目標:**
- 今読んでいる中国武侠小説の翻訳が完結するまで追いかける
- 日本語の原作を少しずつ読めるようになりたい（翻訳と原文を比較しながら）

**長期的な目標:**
- 世界中の小説を言語の壁なく楽しめる環境がほしい
- 好きな作品の翻訳品質について、コミュニティでフィードバックを共有する仕組み

**潜在的なニーズ:**
- 翻訳の「品質レベル」が事前に分かる仕組み（AI翻訳 / 人力翻訳 / 監修済み等）
- 原文と翻訳を並べて読める機能（語学学習にも活用したい）

## 課題・ペインポイント

**主な課題:**

1. **WebNovelの翻訳品質が低すぎて読書体験が損なわれる** - 名前が章の途中で変わる、代名詞のミス、文脈が失われる。「読めないレベル」のAI翻訳も混在
2. **課金が不透明で高額** - WebNovelで1作品を完結まで読むと$50〜$200以上かかることがある。コイン制で実際の金額が分かりにくい
3. **質の高い翻訳作品を見つけるのが困難** - AI生成コンテンツ（AIスロップ）が増え、良作が埋もれている

**日常的なフラストレーション:**
- 翻訳小説を読んでいて「これは何を言っているんだ？」と思う瞬間が頻繁にある
- Wattpadは広告が多すぎて読書に集中できない
- 好きな作品の翻訳が突然打ち切りになることがある（翻訳者が離脱）

**制約条件:**
- 月の読書予算は$30程度（Netflix等のサブスク感覚）
- 日本語・中国語の原文を直接読むには語学力が不足
- 通勤中はスマートフォンでしか読めない（片手操作が必要）

## 購買行動・意思決定

**購買の動機:**
- 続きが気になる作品のためなら課金する（ただし価格の透明性が前提）
- 「広告なし」の読書体験に対して月額課金を検討する意欲がある

**意思決定のプロセス:**

1. Redditやコミュニティで作品の評判を確認する
2. 無料の序盤を読んでみて翻訳品質と作品の質を判断する
3. 翻訳品質が許容ラインを超えていれば課金を検討する

**重視するポイント:**
- 翻訳品質（固有名詞の一貫性、自然な文体、文脈の維持）
- 料金の透明性（いくら払えば何が読めるか明確にわかること）
- 作品ディスカバリー（好みに合った新作を効率的に見つけられること）

**購買の障壁:**
- 翻訳品質が不明な状態で課金するのは抵抗がある
- コイン制のような不透明な課金体系は不信感を持つ
- 過去にWebNovelで「気づいたら大金を使っていた」経験がありトラウマ

**影響を受ける情報源:**
- Reddit（r/noveltranslations）の推薦スレッド
- NovelUpdatesのレビューとランキング
- Discord翻訳コミュニティのメンバーの評価

## プロダクトとの関わり

**利用シーン:**
- 通勤中のバス内でスマートフォンで1〜2章読む（朝・夕の各40分）
- 就寝前にベッドでiPadで3〜5章まとめ読み
- 週末のカフェで半日かけて溜まった更新分を一気読み

**利用頻度:** 毎日（平均1.5〜2時間/日）

**期待する価値:**
- 翻訳であることを忘れるほど自然な日本語↔英語の翻訳品質
- 月額定額で好きなだけ読める透明な料金モデル
- 東アジアの小説を中心とした豊富なカタログ

**理想的な体験:**
- アプリを開く → レコメンドで好みに合った新作を発見 → 翻訳品質に違和感なく没頭して読む → 気に入ったら作家に直接「投げ銭」で支援する
