## 目標・ニーズ

**達成したい目標:**

1. 自分の作品を海外の読者にも届けたい（特に英語圏・中国語圏）
2. 執筆活動だけで生計を立てられるレベルの収益を得たい
3. 書籍化だけでなく、デジタルプラットフォームでの新しいマネタイズ手段を見つけたい
4. プロット管理・キャラクター設定・執筆・投稿がすべて1つの環境で完結する「理想の執筆体験」を手に入れたい

**短期的な目標:**
- 現在連載中の4作目を完結させる（残り約30話）
- 海外読者からの反応を得たい（まずは英語翻訳で試してみたい）

**長期的な目標:**
- 海外読者からの収益が月10万円を超える状態
- 多言語で同時連載できる環境を確立する
- プロ作家としての独立

**潜在的なニーズ:**
- 翻訳された自分の作品の品質を確認・コントロールする手段
- 言語圏ごとの読者の反応を分析できるダッシュボード
- 翻訳版の著作権管理と収益分配の透明な仕組み
- プロット管理・設定資料・原稿執筆・投稿が一気通貫で行える統合執筆環境
- 執筆中にキャラクター設定や世界観設定をサイドパネルで即参照できる機能

## 課題・ペインポイント

**主な課題:**

1. **翻訳コストが高すぎる** - プロの翻訳者に依頼すると1話あたり数万円。100話超の長編では現実的に不可能
2. **カクヨムの収益が少ない** - 150万PVでも月3〜5万円程度のリワード。海外読者へのリーチ手段がない
3. **AI翻訳の品質が信用できない** - DeepLで自作を翻訳してみたが、ファンタジー特有の用語（魔法名、固有名詞）が不自然で、自分の作品として公開する気になれない
4. **執筆環境が分断されている** - Notionでプロット管理、VS Codeで原稿執筆、カクヨムに手動コピペで投稿という3ツール横断のワークフロー。キャラクター設定を確認するたびにウィンドウを切り替え、投稿時にフォーマットが崩れることもある

**日常的なフラストレーション:**
- 「英語版ないの？」というコメントに「ない」としか答えられないもどかしさ
- カクヨムのリワード制度だけでは生活できない
- WebNovelの作家搾取（収益80%取得）を聞いて、海外プラットフォームへの不信感がある
- Notionのプロット → VS Codeの原稿 → カクヨムの投稿画面を行き来する非効率さ。「書く」以外の作業に時間を取られる
- カクヨムのエディタが貧弱で、VS Codeで書いてからコピペする二度手間
- 長編の設定管理（キャラクター100人超、魔法体系、世界地図）がNotionのページに散らばり、執筆中に参照しにくい

**制約条件:**
- 翻訳に使える予算は月1万円程度が限界
- 本業があるため、翻訳作業に割ける時間はほぼゼロ
- 英語力はTOEIC 650程度。翻訳品質を自分で検証するのが難しい

## 購買行動・意思決定

**購買の動機:**
- 自作品を海外に届けるための「投資」として考える
- 収益が見込めるプラットフォームには積極的に参加したい

**意思決定のプロセス:**

1. X.comの作家コミュニティで新プラットフォームの評判を確認する
2. 収益分配の条件（作家側の取り分）を最初にチェックする
3. 他の作家の体験談を読んでからアカウント作成を検討する

**重視するポイント:**
- 収益分配の公正さ（作家の取り分が50%以上あること）
- 著作権がプラットフォームに帰属しないこと
- 翻訳品質を自分でコントロールできる仕組み

**購買の障壁:**
- WebNovelの「著作権帰属」問題を知っており、権利関係に非常に敏感
- 新しいプラットフォームに作品を移行する手間（既存読者を失うリスク）
- 翻訳品質が悪いと自分の作品のブランドが傷つくリスク

**影響を受ける情報源:**
- X.comの作家コミュニティ（信頼する先輩作家の意見）
- カクヨム・なろう作家のブログ記事
- 書籍化経験のある作家のインタビュー記事

## プロダクトとの関わり

**利用シーン:**
- 新章を書き上げた直後に翻訳プレビューを確認する
- 週1回、翻訳版の読者コメント・反応をチェックする
- 月1回、収益ダッシュボードで言語圏別の実績を確認する

**利用頻度:** 週3〜5回（執筆ペースに連動）

**期待する価値:**
- プロット管理から執筆・投稿・翻訳までがシームレスに繋がる統合執筆環境
- 執筆中にキャラクター設定・世界観設定をサイドパネルで即参照できる快適さ
- ワンクリックで作品が高品質に多言語翻訳される体験
- 翻訳結果の固有名詞辞書を自分でカスタマイズできる機能
- 公正な収益分配（最低でも50%以上が作家に）

**理想的な体験:**
- 統合エディタでプロットを確認しながら原稿を書く → 書き終えたら投稿ボタンを押すだけ → 自動で英語・中国語等に高品質翻訳される → 海外読者からの反応がリアルタイムで届く → 翻訳版からの収益も透明に確認できる
