プロダクトが提供する主要な機能（作家、読者、翻訳）を定義します。

## 作家向け機能 (Writer Features)

### アカウント・設定

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **ユーザー登録/ログイン** | メール/パスワード、Google認証。作家/読者のロール選択。 | Must | 1 |
| **プロフィール管理** | アイコン、ペンネーム、自己紹介（多言語対応）。 | Must | 1 |
| **ダッシュボード** | 作品ごとの閲覧数、ブックマーク数、収益などの統計表示。 | Should | 2 |

### 執筆・投稿（Novel Studio）

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **作品管理** | 作品の新規作成、編集、削除。表紙画像設定。 | Must | 1 |
| **章管理** | 章の追加、並べ替え、下書き保存、公開設定。 | Must | 1 |
| **高機能エディタ** | 縦書き/横書きプレビュー、ルビ入力支援、自動保存。 | Must | 1 |
| **用語集管理** | キャラクター名や独自用語の翻訳ルール設定。 | Should | 2 |
| **予約投稿** | 指定した日時に章を公開する機能。 | Should | 2 |

### 収益化・その他

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **コメント管理** | 読者からのコメントへの返信、非表示、通報。 | Should | 2 |
| **収益管理** | 広告収益、投げ銭、サブスク収益の確認と振込申請。 | Should | 3 |
| **お知らせ機能** | フォロワーへの一斉通知放送。 | Want | 3 |

## 読者向け機能 (Reader Features)

### 閲覧体験 (Reading Experience)

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **多言語ビューア** | 原文/翻訳文の切り替え、並列表示、フォント/背景色設定。 | Must | 1 |
| **ルビ表示** | 原文のルビを表示/非表示、ツールチップで意味表示。 | Should | 2 |
| **オフライン閲覧** | アプリ版でのダウンロード機能（プレミアム限定）。 | Want | 3 |
| **縦書き/横書き切替** | 日本語原文の場合の縦書き表示対応。 | Must | 1 |

### 発見・ライブラリ

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **作品検索** | ジャンル、タグ、人気順、新着順での検索・フィルタリング。 | Must | 1 |
| **ランキング** | 総合、ジャンル別、急上昇ランキング。 | Should | 2 |
| **本棚（ライブラリ）** | 読んでいる作品、読了作品、あとで読むリストの管理。 | Must | 1 |
| **履歴** | 閲覧履歴と続きからの再生。 | Must | 1 |

### コミュニティ・応援

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **コメント投稿** | 章ごとのコメント投稿、いいね、返信。 | Must | 2 |
| **レビュー** | 作品全体への星評価とレビュー投稿。 | Should | 2 |
| **フォロー** | お気に入り作家のフォローと更新通知。 | Must | 2 |
| **投げ銭 (Tip)** | 作家への直接支援機能。 | Want | 3 |

## 自動翻訳・用語集機能 (Translation Features)

### 翻訳エンジン・パイプライン

Astranote の中核となる機能です。

| 機能名 | 概要 | 優先度 | Phase |
| ------ | ---- | ------ | ----- |
| **AI自動翻訳** | 投稿と同時に日→英（他言語）へ自動翻訳。Claude 3.5 Sonnet / GPT-4使用。 | Must | 1 |
| **ルビ処理** | 日本語のルビを認識し、適切に翻訳または削除して処理。 | Must | 1 |
| **用語集適用** | 作家が登録した用語集に基づき、固有名詞を一貫して翻訳。 | Must | 2 |
| **文脈維持** | 前後の文脈を考慮した翻訳プロンプト。 | Should | 1 |

### 言語対応フェーズ

1. **Phase 1**: 英語 (English) - 最優先
2. **Phase 2**: 韓国語 (Korean), 中国語繁体字 (Traditional Chinese)
3. **Phase 3**: 中国語簡体字, スペイン語
4. **Phase 4**: フランス語, ドイツ語, その他

## 非機能要件

- **パフォーマンス**: ページロード 1.5秒以内、翻訳処理 1章あたり30秒以内
- **可用性**: 稼働率 99.9%
- **セキュリティ**: ユーザーデータの暗号化、不正アクセス防止
- **SEO**: 各言語での検索エンジン最適化、OGP自動生成
