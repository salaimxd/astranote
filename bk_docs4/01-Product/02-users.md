## ターゲットセグメント

### プライマリ: 日本のライトノベル作家

| 属性 | 詳細 |
|------|------|
| 年齢層 | 18-35歳 |
| 志向 | 異世界・ファンタジー中心、出版デビュー狙い |
| 現在の活動場所 | 小説家になろう、カクヨム |
| 推定規模 | 約10万人（アクティブWeb小説作家） |

**ペイン（課題）:**
- 国内市場での埋もれ、差別化の難しさ
- 海外読者へのリーチ手段がない
- 翻訳コストが高く個人では非現実的

**ゲイン（得られる価値）:**
- グローバル読者獲得による差別化
- 海外からの新たな収益機会
- 世界中のファンコミュニティ形成

### セカンダリ: 海外のラノベファン

| 属性 | 詳細 |
|------|------|
| 地域 | 北米、韓国、台湾、欧州 |
| 年齢層 | 18-30歳 |
| 志向 | 日本のラノベ・アニメ文化のファン |
| 推定規模 | 約500万人（グローバルラノベ読者） |

**ペイン（課題）:**

- 翻訳されない作品が多い
- 翻訳品質にばらつきがある

**ゲイン（得られる価値）:**
- 最新作にすぐアクセスできる
- 母国語で快適に読める
- 作家と直接つながれる

## ペルソナ

### ペルソナ1: 山田太郎（作家志望）

| 項目 | 内容 |
|------|------|
| 年齢・性別 | 28歳、男性 |
| 職業 | ITエンジニア（会社員） |
| 居住地 | 東京都 |
| 執筆歴 | 3年 |
| 現在の活動 | 小説家になろうで異世界ファンタジーを連載中 |

**執筆状況:**
- 週2-3話ペースで更新
- フォロワー数: 約2,000人
- 書籍化経験: なし（狙っている）

**課題:**
- 国内ランキングで埋もれがち
- 差別化が難しい（似た作品が多い）
- 海外読者に届ける手段がない

**Astranote への期待:**
- 海外読者を獲得して差別化したい
- グローバルでの評価を国内出版社にアピールしたい
- 翻訳なしで世界に発信できるのは魅力的

**利用シナリオ:**
1. 既存作品を Astranote にも投稿
2. 海外読者からのコメント・反応を確認
3. 人気が出れば Astranote をメインプラットフォームに
