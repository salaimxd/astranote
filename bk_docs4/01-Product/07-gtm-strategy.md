プレローンチ、ローンチ、スケーリングの3フェーズに分けた市場投入計画を定義します。

## エグゼクティブサマリー

### コアメッセージ

> **「あなたの物語を、世界へ。」**
> 日本語で書くだけで、世界中の読者に届く。

### 差別化ポイント

- **唯一無二**: グローバル × ラノベ特化 × AI自動翻訳
- **競合不在**: このポジションに競合がいない

## Phase 1: プレローンチ (クローズドβ)

**期間**: ローンチの2-4週間前

### 目標

- プロダクトの検証とフィードバック収集
- 初期コンテンツの確保（招待作家50-100名）
- ローンチ時のアンバサダー育成

### 主要アクティビティ

| アクティビティ | 担当 | 期限 | ステータス |
| -------------- | ---- | ---- | ---------- |
| ターゲット作家リスト作成（200名） | マーケティング | D-30 | 未着手 |
| 招待DM送信・フォローアップ | マーケティング | D-14 | 未着手 |
| βテスターコミュニティ運営 | コミュニティ | D-14 | 未着手 |

### チャネル

- Twitter/X (DMによる直接勧誘)
- なろう/カクヨム (活動報告での告知依頼)

### オファー（βテスター特典）

- 永久プレミアム会員（Proプラン永久無料）
- ローンチ時の優先露出
- 開発チームへの直接フィードバック権

## Phase 2: ローンチ

**期間**: ローンチ日〜1ヶ月

### 目標

- 作家数 500人
- 読者 MAU 5,000人
- 翻訳利用率 80%

### ローンチ日アクティビティ

- [ ] プレスリリース配信 (PR TIMES等)
- [ ] Twitter/X でのローンチ告知
- [ ] インフルエンサーによる拡散
- [ ] β作家による一斉告知

### チャネル

| チャネル | アプローチ | 予算 |
| -------- | ---------- | ---- |
| プレスリリース | PR TIMES, IT/アニメメディア | 3万円 |
| Twitter/X | 公式アカウント運用, ハッシュタグCP | 人件費のみ |
| Reddit | r/lightnovels 等での告知 | 人件費のみ |

### ローンチキャンペーン

**「世界デビューキャンペーン」**:
期間中に投稿された作品は、通常英語のみ無料のところ、全言語翻訳を無料で提供。初期コンテンツの充実を図る。

## Phase 3: スケーリング

**期間**: ローンチ後3ヶ月〜

### 目標

- 作家数 5,000人
- 読者 MAU 50,000人
- 海外読者比率 50%以上

### スケーリング戦略

#### 優先順位付け

| チャネル | 現在の成果 | 可能性 | 優先度 |
| -------- | ---------- | ------ | ------ |
| Twitter (作家獲得) | 高 | 中 | 高 |
| SEO (読者獲得) | 低 | 高 | 高 |
| Reddit (読者獲得) | 中 | 高 | 高 |
| 有料広告 | - | 高 | 中 |

#### 成長実験

| 実験 | 仮説 | 測定指標 | 期間 |
| ---- | ---- | -------- | ---- |
| 「翻訳チャレンジ」コンテスト | 賞品があれば高品質な短編が集まる | 投稿数、翻訳精度 | 2週間 |
| リファラルプログラム | 特典があれば作家が作家を呼ぶ | 招待数 | 継続 |
