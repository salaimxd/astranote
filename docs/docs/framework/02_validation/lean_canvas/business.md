## 6. 収益の流れ（Revenue Streams）

**推奨記入順序: 6番目** --- ビジネスの持続可能性を設計する。

### 価格設定

| 収益源 | 対象 | 価格 | 説明 |
|--------|------|------|------|
| ライター月額プラン | 作家 | ¥1,000/月（約$7） | AI翻訳機能、エディタ、公開機能を含むSaaSサブスクリプション |
| 読者月額プラン | 読者 | $5/月 | 全翻訳作品の読み放題サブスクリプション |
| 読者無料プラン | 読者 | 無料（広告付き） | 広告表示ありで一部作品を閲覧可能 |
| 投げ銭手数料 | 読者→作家 | 30%手数料（70%作家還元） | 読者が作家に直接支援するトランザクション手数料 |
| 広告収入 | 広告主 | CPM/CPC | 無料プラン読者への広告表示 |

### 収益モデル

- **SaaS型**: 作家のサブスクリプション（月額定額）
- **マーケットプレイス型**: 読者のサブスクリプション（プラットフォーム利用料）
- **トランザクション型**: 投げ銭の手数料収入（30%）
- **広告型**: 無料プラン読者への広告表示

### 想定LTV（顧客生涯価値）

| セグメント | 月額 | 想定継続月数 | LTV |
|-----------|------|-------------|-----|
| ライター（作家） | ¥1,000 | 24ヶ月 | ¥24,000（約$160） |
| 読者（有料） | $5 | 18ヶ月 | $90 |
| 読者（無料・広告） | 広告CPM $3想定 | 12ヶ月 | $3.6 |

### ユニットエコノミクスの概算

> **注記（2026-02-17修正）:** 旧版のAI API費用¥300は過大見積もりのため修正。GPT-4o使用・月10話翻訳（50,000文字）想定で¥25〜75に再計算。これにより月間粗利は上方修正となる。損益分岐点は下記の損益分岐点セクションで再計算済み。

| 指標 | ライター | 読者（有料） |
|------|---------|-------------|
| LTV | ¥24,000（約$160） | $90 |
| 目標CAC | ¥8,000（約$53） | $15 |
| LTV/CAC比率 | 3.0x | 6.0x |
| 月間売上 | ¥1,000 | $5（≈¥750） |
| 月間変動費（API・インフラ） | ¥25〜75（API: GPT-4o・10話/月）+ ¥200（インフラ配分） = ¥225〜275 | ¥100（サーバー・CDN配分） |
| 月間粗利（修正後） | ¥725〜775（中央値: 約¥750） | 約¥650（約$4.3） |
| 回収期間（修正後） | 約10.7ヶ月（¥8,000 ÷ ¥750） | 約3.5ヶ月（$15 ÷ $4.3） |

**損益分岐点への影響:**

- ライター1人あたり月間粗利が旧版¥700→修正後¥750に改善（変動費の適正化による）
- 損益分岐点は下記「損益分岐点の概算」セクションで修正値を反映

## 7. コスト構造（Cost Structure）

**推奨記入順序: 7番目** --- 必要なコストを把握し、損益分岐点を見積もる。

### 固定費

| 費目 | 月額概算 | 備考 |
|------|---------|------|
| 開発人件費 | ¥1,500,000 | エンジニア2名（フルタイム換算） |
| サーバー/インフラ | ¥100,000 | AWS/GCP、CDN、データベース |
| AI API基本料 | ¥50,000 | 翻訳エンジンの基本利用料 |
| ドメイン・SaaS | ¥30,000 | ドメイン、メール、分析ツール等 |
| **固定費合計** | **¥1,680,000/月** | |

### APIモデル選定とコスト比較

翻訳品質とコストのトレードオフを考慮し、利用するAIモデルを明確に選定する。

| モデル | コスト目安 | 1話（5,000文字）あたり | 10話/月のAPI費用 | 特徴 |
|--------|-----------|----------------------|----------------|------|
| Claude Opus | 約¥12.6/1,000文字 | 約¥63 | 約¥630 | 最高品質。ラノベ文脈理解が強力だが高コスト |
| GPT-4o | 約¥0.5〜1.5/1,000文字 | 約¥2.5〜7.5 | 約¥25〜75 | 品質・コストバランスが良い。推奨候補 |
| Claude Haiku | 約¥0.2〜0.5/1,000文字 | 約¥1〜2.5 | 約¥10〜25 | 低コスト。定型的な翻訳に適する |

**推奨モデル（MVP）: GPT-4o**

- 根拠: ラノベ翻訳に必要な文脈理解力と、ライター月額¥1,000の収益モデルに収まる変動費のバランスが最も優れる
- API費用はライター1人あたり月¥25〜75（10話翻訳想定）。月額¥1,000の収益に対して変動費比率は2.5〜7.5%
- 翻訳品質が要件を満たさない場合は Claude Opus への切り替えを検討（コスト構造の見直しが必要）
- MVP後の品質フィードバックデータを蓄積後、モデル選定を再評価する

### 変動費

| 費目 | 単価 | 備考 |

| 決済手数料（Stripe） | 3.6% | 投げ銭・サブスクリプション決済 |
| 帯域/配信コスト | ¥0.01/リクエスト | 作品配信のCDNコスト |

| チャネル | 想定CAC | 備考 |
|----------|---------|------|
| オーガニック（SEO/コミュニティ） | ¥0〜500 | 初期の主要チャネル |

### 損益分岐点の概算

> **注記（2026-02-17修正）:** 旧版のAPI変動費¥300/ライターを修正し、GPT-4o・月10話翻訳想定で¥25〜75に再計算。月間粗利の改善を受けて損益分岐点も更新した。

| 項目 | 旧値 | 修正後 | 変更内容 |
|------|------|--------|---------|
| 月間固定費 | ¥1,680,000 | ¥1,680,000 | 変更なし |
| ライター1人あたり月間粗利 | ¥700 | **¥750**（中央値） | API費用の適正化（¥300→¥250）により改善 |
| 読者1人あたり月間粗利 | ¥600（$4相当） | **¥650**（約$4.3） | サーバー費配分の見直し |
| 損益分岐点（ライターのみ） | 約2,400名 | **約2,240名** | ¥1,680,000 ÷ ¥750 |
| 損益分岐点（ライター:読者=1:5の場合） | ライター400名+読者2,000名 | **ライター373名+読者1,865名** | 粗利改善により若干縮小 |
| 目標到達時期 | 18〜24ヶ月 | 18〜24ヶ月 | 変更なし |

**注意:** 上記は推定値であり、実際のAPIコスト・翻訳量・ユーザー行動によって変動する。MVPフェーズで実測値を取得後、再計算する。

## 8. 主要指標（Key Metrics）

**推奨記入順序: 8番目** --- 成功を測る指標を定義する。

### AARRR各段階の指標と目標値

| ステージ | 指標 | 目標値（6ヶ月後） | 目標値（12ヶ月後） |
|----------|------|-------------------|-------------------|
| **Acquisition（獲得）** | 月間サイトUU | 10,000 UU | 50,000 UU |
| | 作家登録数 | 200名 | 1,000名 |
| | 読者登録数 | 2,000名 | 15,000名 |
| **Activation（活性化）** | 作家の初回作品公開率 | 40% | 50% |
| | 読者の初回3話完読率 | 30% | 40% |
| | 翻訳機能利用率（作家） | 60% | 75% |
| **Retention（継続）** | 作家の月間リテンション率 | 70% | 80% |
| | 読者の月間リテンション率 | 50% | 65% |
| | 週間アクティブ作家率 | 40% | 55% |
| **Revenue（収益）** | MRR（月間定期収益） | ¥500,000 | ¥3,000,000 |
| | 有料読者転換率 | 5% | 10% |
| | 投げ銭利用率 | 3% | 8% |
| **Referral（紹介）** | 紹介経由の登録率 | 10% | 20% |
| | SNSシェア率（作品単位） | 5% | 15% |

### 最も重要な3つの指標

1. **翻訳機能利用率（作家）**: astranoteの核心的価値。作家が翻訳機能を使わなければ、プラットフォームの存在意義がない。目標: 75%
2. **読者の月間リテンション率**: 読者が継続的に利用するかが、作家への還元とプラットフォームの持続可能性を決定する。目標: 65%
3. **MRR（月間定期収益）**: ビジネスの持続可能性を直接示す指標。損益分岐点（¥1,680,000/月）到達が最重要マイルストーン。目標: 12ヶ月後に¥3,000,000
