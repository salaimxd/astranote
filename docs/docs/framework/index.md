プロダクト開発のフレームワークへようこそ。このガイドは、アイデアの発見から拡大まで、プロダクト開発の全プロセスをサポートします。

## フレームワークの構成

### 🔍 01. Discovery（発見）

ユーザーのニーズを深く理解し、解決すべき課題を明確化します。

- [ユーザーリサーチ](/framework/01_discovery/user_research) - ユーザーの生の声を収集
- [競合分析](/framework/01_discovery/competitive_analysis) - 競合の強み・弱みを理解
- [ペルソナ](/framework/01_discovery/persona) - ターゲットユーザーの具体化
- [共感マップ](/framework/01_discovery/empathy_map) - ユーザーの思考と感情の可視化
- [ジャーニーマップ](/framework/01_discovery/journey_map) - ユーザー体験の全体像
- [HMW](/framework/01_discovery/hmw) - 問題を機会に変える問いの作成

### ✅ 02. Validation（検証）

アイデアの実現可能性とビジネスモデルを検証します。

- [リーンキャンバス](/framework/02_validation/lean_canvas) - ビジネスモデルの設計
- [MVP](/framework/02_validation/mvp) - 最小限の機能で価値を検証

### 🎯 03. Refinement（深化）

プロダクトの体験を詳細に設計し、開発に向けて準備します。

- [ジャーニーマップ](/framework/01_discovery/journey_map) - ユーザー体験の時系列可視化（Discovery と共有）
- [ユーザーストーリー](/framework/03_refinement/user_stories) - 機能を価値の視点で記述
- [ストーリーマッピング](/framework/03_refinement/story_mapping) - 機能の優先順位付け
- [テストケース](/framework/03_refinement/test_cases) - 品質保証の計画

### 🧭 04. Strategy（戦略）

なぜこのプロダクトを作るのか、ビジネスモデルと成長指標を定義します。

- [価値提案](/framework/04_strategy/value_proposition) - 誰の、どんな苦痛を、どう取り除くか
- [BMC](/framework/04_strategy/bmc) - ビジネスモデルキャンバス
- [North Star Metric](/framework/04_strategy/northstar) - 成長の指標

### 📦 05. Specification（仕様）

何を作るのかを具体的に定義します。

- [PRD](/framework/05_specification/prd) - 要件定義書
- [UI Flows](/framework/05_specification/ui_flows) - 画面遷移図
- [Design System](/framework/05_specification/design_system) - UIガイドライン

### ⚙️ 06. Engineering（技術）

どう作るのかを技術的に設計します。

- [Tech Stack](/framework/06_engineering/tech_stack) - 技術選定
- [Architecture](/framework/06_engineering/architecture) - システム構成図
- [API Docs](/framework/06_engineering/api_docs) - API仕様書
- [ADR](/framework/06_engineering/adr) - アーキテクチャ決定記録
- [データ基盤](/framework/06_engineering/data_infra) - データドリブンな意思決定

### 🚀 07. Go-to-Market（市場投入）

どう届けるのかを戦略的に設計します。

- [4P分析](/framework/07_go_to_market/4p_analysis) - マーケティング戦略の設計
- [Distribution Channels](/framework/07_go_to_market/distribution_channels) - 流通経路
- [Launch Checklist](/framework/07_go_to_market/launch_checklist) - リリース手順書
- [ASO/SEO Strategy](/framework/07_go_to_market/aso_seo_strategy) - 検索対策

### 📈 08. Metrics（計測）

うまくいっているかを数値で測定します。

- [KPI](/framework/08_metrics/kpi) - 重要業績評価指標
- [Funnel Analysis](/framework/08_metrics/funnel_analysis) - ファネル分析
- [Cohort Analysis](/framework/08_metrics/cohort_analysis) - コホート分析
