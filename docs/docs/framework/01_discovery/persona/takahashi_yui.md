## 背景・ライフスタイル

**学歴:** 日本大学文学部卒業

**経歴:** 大学時代にラノベ・Web小説にハマる → 社会人になってからも通勤時間と就寝前の読書が日課 → なろう歴8年、カクヨム歴5年のヘビーリーダー

**ライフスタイル:**

- 平日は通勤時間（片道45分）と就寝前（22時〜24時）が読書タイム
- 週末はカフェで半日読書（3〜5時間）することも
- 月に30〜50話（10〜15作品を並行して追いかけている）
- 年間読書量はWeb小説だけで300〜500話以上

**趣味・興味:**

- Web小説読書（異世界ファンタジー、悪役令嬢もの、追放ざまぁ系が好み）
- アニメ鑑賞（原作がなろう系のアニメを優先的に視聴）
- 読書感想をnoteに投稿（フォロワー200人程度）
- 書籍化された作品のコミカライズも追いかける

**価値観:**

- 「良い作品は正当に評価されるべき」と考えており、気に入った作品は書籍版も購入する
- 読書体験の快適さにこだわりがあり、フォントや背景色の設定に時間をかける
- 作家への応援コメントを積極的に送るタイプ

## 目標・ニーズ

**達成したい目標:**

1. 追いかけている作品の最新話を快適に読みたい
2. 120万作品の中から自分好みの「当たり」作品を効率的に見つけたい
3. 好きな作家を直接応援したい（コメント、投げ銭）

**短期的な目標:** 今期アニメ化された3作品の原作を最新話まで追いつく

**長期的な目標:** 自分の読書感想ブログを通じて、埋もれた良作を発掘して広めたい

**潜在的なニーズ:**

- 読書体験そのものの質を上げたい（縦書き、目に優しい配色、しおり機能の充実）
- 海外で人気のある日本の作品を知りたい（逆輸入的な発見）

## 課題・ペインポイント

**主な課題:**

1. **検索・発見が絶望的に困難** — なろうアプリに検索機能がない。120万作品の中から読みたい作品を見つけるために、ブラウザで検索 → ブックマーク → アプリで読むという二重作業を強いられる
2. **読書体験の中断** — バッテリー異常消費（1時間で20%以上減る）、アプリクラッシュ、読書位置のリセットにより没入感が壊される
3. **日本語表示の最適化不足** — 縦書き未対応、フォント選択肢が少ない、行間・余白の調整ができない。「紙の本のような読書体験」には程遠い

**日常的なフラストレーション:**

- 通勤電車で読んでいる最中にアプリがクラッシュし、読んでいた箇所を見失う
- なろうのランキングは上位が固定化しており、新しい良作を見つけにくい
- カクヨムアプリの「ブラウザの方がマシ」という体験。アプリをインストールした意味を感じない
- 広告のせいでUIの位置がコロコロ変わり、誤タップでストレスが溜まる

**制約条件:**

- 読書に使える予算は月3,000〜5,000円程度（書籍購入含む）
- 通勤中はスマートフォンのみ。片手操作が前提
- ITリテラシーが高くないため、複雑な設定や外部ツール連携は苦手

## 購買行動・意思決定

**購買の動機:** 「読書体験が明らかに良くなる」と実感できれば課金する。気に入った作品の書籍版は必ず買う

**意思決定のプロセス:**

1. X (Twitter) やnoteで評判を確認
2. 無料で試してみて、使い勝手を体感
3. 1〜2週間使い続けて「前のアプリに戻りたくない」と感じたら定着

**重視するポイント:**

- 読書体験の快適さ（縦書き、フォント、バッテリー消費、安定性）
- 作品のラインナップ（追いかけている作品が読めるか）
- 操作のシンプルさ（複雑な機能より「すぐ読める」が大事）

**購買の障壁:**

- 「なろうで無料で読めるのに、わざわざ別のアプリにお金を払う理由がない」
- 追いかけている作品がastranoteに移らない限り、乗り換える動機がない
- 新しいアプリの学習コスト（慣れたUIから離れる心理的抵抗）

**影響を受ける情報源:**

- X (Twitter) のフォロー作家が「astranoteに移行しました」と告知
- noteの読書感想で「読みやすさが段違い」というレビュー
- 友人からのLINEでの口コミ

## プロダクトとの関わり

**利用シーン:**

- 通勤電車（iPhone、片手操作）で更新された作品の最新話を読む
- 就寝前（iPhone、暗い部屋）でダークモードで読書
- 週末カフェ（PC）で新しい作品を探索・発見

**利用頻度:** 毎日、1回あたり30分〜2時間

**期待する価値:**

- 途切れない没入型読書体験（バッテリー消費が少なく、クラッシュしない）
- 効率的な作品発見（検索、フィルタ、AIレコメンド）
- 日本語に最適化された表示（縦書き、豊富なフォント設定）
- 作家への直接応援（投げ銭、コメント）

**理想的な体験:**

- アプリを開いたら、フォロー中の作品の新着が一覧表示され、ワンタップで読み始められる。縦書き・明朝体で紙の本のような読み心地。読んだ後は作家にスタンプや投げ銭で応援。週末は「あなたへのおすすめ」から未知の良作を発掘する。
