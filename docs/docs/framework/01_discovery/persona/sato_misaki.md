## 目標・ニーズ

**達成したい目標:**

1. Web小説の収益を増やし、ライター仕事を減らして創作に集中したい
2. 海外ファンにも自分の作品を届けたい（英語圏・中国語圏）
3. ファンからの直接的な応援（投げ銭等）を受けたい

**短期的な目標:** 連載中の作品（3作品同時連載中）のうち1つを新たに書籍化にこぎつける

**長期的な目標:** Web小説家として独立（ライター仕事をゼロにする）。海外でもファンベースを持つ

**潜在的なニーズ:**

- 海外の読者が自分の作品をどう感じているか知りたい。異文化の読者の反応に興味がある

## 課題・ペインポイント

**主な課題:**

1. 書籍化しても印税率が低い（通常8〜10%、2巻で約50万円の印税）
2. 海外展開の方法が分からない（出版社頼みだが、中堅作家の優先度は低い）
3. 「なろう」「カクヨム」に3作品を同時投稿する作業負荷が大きい

**日常的なフラストレーション:**

- 同じ原稿を「なろう」「カクヨム」の2サイトにコピペ投稿する手間
- 書籍化の印税が入ってきても生活するには不十分（ライター仕事を辞められない）
- ファンから「英語版はないの？」とたまに聞かれるが、対応できない

**制約条件:**

- 翻訳に使える予算はある程度ある（月5,000円程度なら出せる）
- 技術的なことは苦手（自分でWebサイトを作るのは無理）
- 時間がない（3作品同時連載 + ライター仕事で毎日16時間稼働）

## 購買行動・意思決定

**購買の動機:** 「収益が増える」「作業効率が上がる」「ファンベースが拡大する」

**意思決定のプロセス:**

1. 出版社の編集者や作家仲間からの推薦
2. 実際の収益化事例やデータを確認
3. 無料プランで試してから有料プランに移行

**重視するポイント:**

- 収益の透明性（いくら稼げるのかが明確）
- 投稿の手軽さ（複数プラットフォームへの同時投稿の手間を減らしたい）
- ファンとの距離感（投げ銭やコメントで直接交流）

**購買の障壁:**

- 「新しいプラットフォームに読者がいるのか」という不安
- 「海外展開して本当に読まれるのか」という懐疑
- 既存の「なろう」「カクヨム」の読者を失うリスク

**影響を受ける情報源:**

- 出版社の編集者の意見
- 同じ書籍化作家仲間の口コミ
- 作家向けnote記事の成功事例

## プロダクトとの関わり

**利用シーン:**

- 午後の執筆タイム（PC）で新エピソードを執筆→AI翻訳→公開
- 夜にスマホで海外読者からのコメント・投げ銭を確認
- 月末に収益レポートをチェック

**利用頻度:** 毎日（執筆・確認）

**期待する価値:**

- 1つのプラットフォームで書くだけで日本語圏+海外に同時に届く
- ファンからの投げ銭で直接収益を得られる
- 執筆→翻訳→公開の流れがシームレス

**理想的な体験:**

- 1つの場所で書くだけで、日本語読者にも海外読者にも届き、両方から投げ銭やサブスクで収益が得られる
