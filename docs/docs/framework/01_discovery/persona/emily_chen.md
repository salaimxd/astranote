## 目標・ニーズ

**達成したい目標:**

1. 好きな日本のWeb小説を、翻訳待ちなしで最新話まで読みたい
2. 翻訳品質が高く、没入感のある読書体験がしたい
3. 好きな作家を直接応援（金銭的に支援）したい

**短期的な目標:** 追いかけている5つのシリーズの最新話に追いつく

**長期的な目標:** 日本語をN2まで上げて原文でも読めるようになりたい。自分でも翻訳活動をしたい

**潜在的なニーズ:**

- 翻訳の待ち時間が長すぎて、他の趣味に時間を使ってしまう。もっと効率よく読書時間を使いたい

## 課題・ペインポイント

**主な課題:**

1. 翻訳が遅い（好きなシリーズが半年以上更新されない、翻訳者が失踪する）
2. 機械翻訳の品質が低く没入できない（Webnovelは「ほぼ読めないレベル」）
3. 課金が高すぎる（Webnovelで月$50以上使うこともある、それでも全話読めない）

**日常的なフラストレーション:**

- Novel Updatesで追いかけているシリーズの翻訳が止まると絶望する
- Webnovelの3層ペイウォール（コイン + メンバーシップ + Privileged Reader）にうんざり
- 日本語のWebサイト（なろう）をDeepLで読もうとするが、ラノベの文体は翻訳が壊れる

**制約条件:**

- 大学院生のため予算は限られている（月$20程度が上限）
- 日本語力はN3レベル（辞書引きながらなら読めるが、楽しめるレベルではない）
- 研究が忙しく、読書に使える時間は週10〜15時間程度

## 購買行動・意思決定

**購買の動機:** 「好きな作品が高品質で読める」「作家に直接お金が届く」なら喜んで払う

**意思決定のプロセス:**

1. Redditやアニメ系Discordでの評判チェック
2. 無料で試し読みして翻訳品質を確認
3. 好きなシリーズがあるかどうかで判断

**重視するポイント:**

- 翻訳品質（ラノベの文脈・ニュアンスが保たれているか）
- コンテンツのラインナップ（好きなジャンル・作家がいるか）
- 価格の妥当性（Kindle Unlimitedの$12/月が基準）

**購買の障壁:**

- 「また別のプラットフォームにお金を払うのか」という疲労感
- 好きなシリーズが移ってこない限り乗り換える理由がない

**影響を受ける情報源:**

- r/lightnovels、r/noveltranslationsのスレッド
- Discordのおすすめチャンネル
- BookTuber / AnimeYouTuberのレビュー

## プロダクトとの関わり

**利用シーン:**

- 通学中のバス（iPhone）で最新エピソードをチェック
- 夜の読書タイム（MacBook / iPad）で長時間読書
- Discordで読んだ作品の感想を共有

**利用頻度:** 週3〜5回、1回あたり1〜2時間

**期待する価値:**

- 翻訳待ちなしで最新話が読める
- 没入感のある高品質翻訳
- 作家に直接投げ銭できる（クリエイター支援）

**理想的な体験:**

- 日本で新しいエピソードが公開されたら、数時間以内に高品質な英語翻訳で読める
