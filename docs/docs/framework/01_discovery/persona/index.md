> このドキュメントは、astranoteのターゲットユーザーを代表する架空の人物像を管理し、チーム全体で共通のユーザーイメージを持つためのセクションです。

## ペルソナとは

ペルソナは、ターゲットユーザーを代表する架空の人物像です。**[ユーザーリサーチ](../user_research.md)で収集した実際の顧客データや調査情報に基づいて作成し**、プロジェクトチーム全体で共通のユーザーイメージを持つために使用します。

## ペルソナ作成のポイント

- **事実に基づく**: ペルソナは想像で作るのではなく、[ユーザーリサーチ](../user_research.md)から得られた事実に基づいて作成する
- **適切な数**: 一般的に3〜5個程度。多すぎるとフォーカスが失われる
- **プライマリペルソナの明確化**: 最重要ターゲットを1つ選定する
- **共感マップとの連携**: それぞれのペルソナに対して個別の[共感マップ](../empathy_map/index.md)を作成する

## ペルソナの作成方法

1. `_template.md` をコピーする
2. ファイル名を `persona_name.md`（例: `tanaka_taro.md`）に変更
3. [ユーザーリサーチ](../user_research.md)の結果を参照しながら記入
4. 上記の「ペルソナ一覧」テーブルに追加
5. `_meta.json` にファイル名を追加
