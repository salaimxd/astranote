## 概要

読者がStar Novelで多言語の小説を快適に閲覧するための機能群です。

:::note[対象ユーザー]
[ユーザー定義](/product/users/) の「グローバル読者」を参照
:::

---

## 作品閲覧

### 機能概要

| 項目 | 内容 |
|------|------|
| 機能ID | NOVEL_READING |
| Phase | 1 |
| 優先度 | Must |

### 言語切り替え

| 項目 | 仕様 |
|------|------|
| 対応言語（Phase 1） | 日本語、英語 |
| 対応言語（Phase 2） | + 韓国語、中国語（繁体） |
| 対応言語（Phase 3） | + 中国語（簡体）、スペイン語 |
| 切り替え方法 | 画面上部のトグル、設定で既定言語設定 |
| 切り替え範囲 | 作品単位/章単位で切り替え可能 |

### 言語切り替えUI

```
┌─────────────────────────────────────────────────────┐
│  📖 異世界転生したら最強だった件                     │
│                                                     │
│  言語: [日本語 ▼]                                   │
│        ┌───────────────────┐                       │
│        │ ○ 日本語（原文）  │                       │
│        │ ● English        │                       │
│        │ ○ 한국어         │                       │
│        │ ○ 繁體中文       │                       │
│        └───────────────────┘                       │
│                                                     │
│  第1章: Beginning                                   │
│  ─────────────────────────────────────────────────  │
│  When I opened my eyes, I found myself in a        │
│  completely different world...                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 閲覧モード

| モード | 説明 |
|--------|------|
| スクロールモード | 縦スクロールで読む（デフォルト） |
| ページモード | 左右フリックでページ送り |
| 横書き/縦書き | 日本語のみ縦書き対応（Phase 2） |

### 表示設定

| 設定 | オプション |
|------|-----------|
| フォントサイズ | 小/中/大/特大（12-24px） |
| フォント | システム標準、明朝、ゴシック |
| 背景色 | 白/セピア/ダーク |
| 行間 | 狭/標準/広 |
| 余白 | 狭/標準/広 |

### ユーザーストーリー

```
読者として、
ワンタップで言語を切り替えられる
ことで、原文と翻訳を比較しながら読める。

読者として、
ダークモードで読める
ことで、夜間でも目に優しく読書できる。
```

## 検索・発見

### 機能概要

| 項目 | 内容 |
|------|------|
| 機能ID | SEARCH_DISCOVERY |
| Phase | 1 |
| 優先度 | Must |

### 検索機能

| 項目 | 仕様 |
|------|------|
| キーワード検索 | タイトル、あらすじ、タグから検索 |
| 対応言語 | 日本語、英語で検索可能 |
| インクリメンタル検索 | 入力中にリアルタイム候補表示 |
| 検索履歴 | 直近20件を保存 |

### フィルタリング

| フィルタ | オプション |
|---------|-----------|
| ジャンル | ファンタジー、SF、ラブコメ等 |
| 連載状態 | 連載中/完結 |
| 文字数 | 短編/中編/長編 |
| 更新日 | 24時間以内/1週間以内/1ヶ月以内 |
| 年齢制限 | 全年齢のみ/R-15含む/すべて |
| 言語 | 翻訳あり/原文のみ |

### ランキング

| ランキング | 集計期間 | 説明 |
|-----------|---------|------|
| 日間 | 24時間 | 直近の人気作 |
| 週間 | 7日間 | 週間人気作 |
| 月間 | 30日間 | 月間人気作 |
| 総合 | 全期間 | 累計人気作 |

### ランキング指標

| 指標 | 重み | 説明 |
|------|------|------|
| PV | 40% | 閲覧数 |
| ブックマーク | 30% | お気に入り数 |
| 評価 | 20% | 平均レート |
| コメント | 10% | コメント数 |

### レコメンド

| 項目 | 仕様 |
|------|------|
| パーソナライズ | 閲覧履歴に基づく推薦 |
| 類似作品 | 「この作品を読んだ人はこちらも」 |
| 新着ピックアップ | 編集部おすすめ |
| ジャンル別おすすめ | ジャンル内の人気作 |

| 項目 | 内容 |
|------|------|
| 機能ID | BOOKMARK_LIBRARY |
| Phase | 1 |
| 優先度 | Must |

### ブックマーク

## フォロー・通知

### 機能概要

| 項目 | 内容 |
|------|------|
| 機能ID | FOLLOW_NOTIFICATION |
| Phase | 2 |
| 優先度 | Should |

### フォロー対象

| 対象 | 説明 |
|------|------|
| 作家 | 作家の新作・更新を追跡 |
| 作品 | 特定作品の更新を追跡 |
| タグ | 特定タグの新作を追跡 |

### 通知設定

| 通知タイプ | オプション |
|-----------|-----------|
| 新章公開 | プッシュ/メール/アプリ内 |
| 新作公開 | プッシュ/メール/アプリ内 |
| 完結 | プッシュ/メール/アプリ内 |
| おすすめ | プッシュ/アプリ内 |

### 通知頻度

| 設定 | 説明 |
|------|------|
| リアルタイム | 即座に通知 |
| 日次ダイジェスト | 1日1回まとめて |
| 週次ダイジェスト | 週1回まとめて |
| オフ | 通知なし |

## コメント・評価

### 機能概要

| 項目 | 内容 |
|------|------|
| 機能ID | COMMENT_RATING |
| Phase | 2 |
| 優先度 | Should |

### コメント

| 項目 | 仕様 |
|------|------|
| 対象 | 作品全体/章ごと |
| 文字数 | 最大1,000文字 |
| 添付 | なし（テキストのみ） |
| ネスト | 返信1階層まで |
| 編集 | 投稿後24時間以内 |
| 削除 | いつでも可能 |

### コメント表示

| 項目 | 仕様 |
|------|------|
| ソート | 新着順/人気順 |
| フィルタ | 言語フィルタ |
| 表示言語 | コメントの言語を自動判定して表示 |
| 翻訳 | コメントの自動翻訳（Phase 3） |

### 評価

| 項目 | 仕様 |
|------|------|
| 形式 | 5段階スター評価 |
| 変更 | いつでも変更可能 |
| 匿名性 | 評価は匿名、レビューは公開 |

### モデレーション

| 機能 | 説明 |
|------|------|
| 報告 | スパム/誹謗中傷/ネタバレを報告 |
| 自動フィルタ | NGワード自動検出 |
| ネタバレ防止 | ネタバレタグ付け機能 |
| ブロック | 特定ユーザーをブロック |

## オフライン閲覧

### 機能概要

| 項目 | 内容 |
|------|------|
| 機能ID | OFFLINE_READING |
| Phase | 3 |
| 優先度 | Could |
| 対象 | プレミアム会員のみ |

### ダウンロード

| 項目 | 仕様 |
|------|------|
| 単位 | 作品単位/章単位 |
| 形式 | 暗号化されたキャッシュ |
| 保存期間 | 30日（オンライン時に更新） |
| 上限 | 100作品まで |

### 同期

| 項目 | 仕様 |
|------|------|
| 自動更新 | Wi-Fi接続時に自動ダウンロード |
| 手動更新 | いつでも手動で更新可能 |
| 読書進捗 | オンライン復帰時に同期 |

## アカウント管理

### 機能概要

| 項目 | 内容 |
|------|------|
| 機能ID | READER_ACCOUNT |
| Phase | 1 |
| 優先度 | Must |

### 登録・ログイン

| 読書進捗同期 | × | ○ |
| コメント | × | ○ |
| フォロー | × | ○ |
| 投げ銭 | × | ○ |

| 項目 | 無料会員 | プレミアム |
|------|---------|-----------|
| 月額 | ¥0 | ¥500 |
| 広告 | あり | なし |
| オフライン閲覧 | × | ○ |
| 先読み | × | 一部作品 |
| バッジ | × | ○ |
