## 概要

このセクションでは、Star Novel で「何を作るのか」を定義します。

:::note[SSOT]
用語の定義は [用語集](/specs/glossary/) を正として参照してください。
:::

---

## 機能カテゴリ

### 作家向け機能

| 機能 | 説明 | Phase |
|------|------|-------|
| アカウント管理 | 登録、ログイン、プロフィール設定 | 1 |
| 作品管理 | 作品作成、章追加、公開設定 | 1 |
| WYSIWYGエディタ | リッチテキストエディタ、ルビ入力対応 | 1 |
| 用語集管理 | 固有名詞・造語の翻訳ルール登録 | 2 |
| アナリティクス | 閲覧数、読者属性、離脱率分析 | 3 |
| 収益ダッシュボード | 投げ銭・サブスク収益確認 | 3 |

### 読者向け機能

| 機能 | 説明 | Phase |
|------|------|-------|
| 作品閲覧 | 多言語での作品閲覧 | 1 |
| ブックマーク | お気に入り作品の保存 | 1 |
| 読書進捗 | 読んだ位置の自動保存 | 1 |
| コメント | 作品・章へのコメント投稿 | 2 |
| フォロー | 作家・作品のフォロー | 2 |
| 投げ銭 | 作家への直接支援 | 3 |
| オフライン閲覧 | ダウンロードして読む（プレミアム） | 3 |

### 翻訳機能

| 機能 | 説明 | Phase |
|------|------|-------|
| 自動翻訳（日→英） | 投稿時に自動で英語翻訳 | 1 |
| 用語集適用 | 登録した用語の一貫した翻訳 | 2 |
| 追加言語（韓・中） | 韓国語、中国語（繁体）対応 | 3 |
| 全言語対応 | 主要言語すべて対応 | 4 |

## 画面一覧

### メイン画面

| 画面ID | 画面名 | 説明 |
|--------|--------|------|
| HOME | ホーム | ランキング、おすすめ、新着 |
| SEARCH | 検索 | 作品検索、フィルタリング |
| NOVEL_DETAIL | 作品詳細 | 作品情報、章一覧、コメント |
| CHAPTER_READ | 読書画面 | 章の本文表示 |
| PROFILE | プロフィール | ユーザー情報、フォロー、履歴 |

### 作家向け画面

| 画面ID | 画面名 | 説明 |
|--------|--------|------|
| DASHBOARD | ダッシュボード | 作品管理、統計概要 |
| NOVEL_EDIT | 作品編集 | タイトル、概要、ジャンル設定 |
| CHAPTER_EDIT | 章編集 | WYSIWYGエディタ |
| GLOSSARY | 用語集 | 翻訳ルール管理 |
| ANALYTICS | 分析 | 詳細統計、読者分析 |

### 認証画面

| 画面ID | 画面名 | 説明 |
|--------|--------|------|
| LOGIN | ログイン | メール/パスワード、OAuth |
| REGISTER | 新規登録 | アカウント作成 |
| FORGOT_PASSWORD | パスワードリセット | メールによるリセット |
