## 概要

カスタマージャーニーマップは、ユーザーがプロダクトを認知してからロイヤルユーザーになるまでの体験を可視化したものです。

Star Novel には2つの主要ユーザータイプがあります：
- **作家（Writer）**: コンテンツを生み出す側
- **読者（Reader）**: コンテンツを消費する側

---

## 作家ジャーニーマップ

### ジャーニー概要

```
認知 → 興味 → 検討 → 登録 → 初回投稿 → 継続利用 → 収益化 → 推奨
```

### Phase 1: 認知（Awareness）

> 「こんなサービスがあるんだ」

#### タッチポイント

| チャネル | コンテンツ | 期待する反応 |
|---------|-----------|-------------|
| Twitter/X | 作家インタビュー記事 | 「面白そう」 |
| なろう/カクヨム | 他作家の活動報告 | 「試してみようかな」 |
| Discord | コミュニティでの言及 | 「詳しく知りたい」 |
| 検索 | 「小説 海外読者」検索結果 | 「解決策があるかも」 |

#### ユーザーの状態

| 項目 | 内容 |
|------|------|
| 思考 | 「自分の作品を海外に届ける方法はないかな」 |
| 感情 | 好奇心、軽い期待 |
| 行動 | SNSで情報収集、サービスページを見る |
| 障壁 | サービスの存在を知らない |

#### 施策

- [ ] Twitter での定期的な作家成功事例投稿
- [ ] なろう/カクヨム人気作家へのアプローチ
- [ ] SEO 対策（「小説 海外読者」「ラノベ 英語翻訳」）

### Phase 2: 興味（Interest）

> 「もっと詳しく知りたい」

#### タッチポイント

| チャネル | コンテンツ | 期待する反応 |
|---------|-----------|-------------|
| LP（ランディングページ） | サービス説明、ベネフィット | 「自分にも使えそう」 |
| ブログ記事 | 「グローバル作家への第一歩」 | 「具体的にわかった」 |
| FAQ | よくある質問 | 「不安が解消された」 |
| 翻訳デモ | サンプル翻訳の確認 | 「品質は十分だ」 |

#### ユーザーの状態

| 項目 | 内容 |
|------|------|
| 思考 | 「本当に自動翻訳で大丈夫？品質は？」 |
| 感情 | 興味と懐疑心が混在 |
| 行動 | LP を読む、翻訳サンプルを確認 |
| 障壁 | 翻訳品質への不安、既存プラットフォームからの移行コスト |

#### 施策

- [ ] 翻訳Before/After のサンプル公開
- [ ] 既存作家の成功事例（数字付き）
- [ ] 「よくある不安」を解消するFAQページ

- [ ] 「並行運用ガイド」の作成
