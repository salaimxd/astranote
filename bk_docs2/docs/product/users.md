## ターゲットセグメント

### プライマリ: 日本のライトノベル作家

| 属性 | 詳細 |
|------|------|
| 年齢層 | 18-35歳 |
| 志向 | 異世界・ファンタジー中心、出版デビュー狙い |
| 現在の活動場所 | 小説家になろう、カクヨム |
| 推定規模 | 約10万人（アクティブWeb小説作家） |

**ペイン（課題）:**
- 国内市場での埋もれ、差別化の難しさ
- 海外読者へのリーチ手段がない
- 翻訳コストが高く個人では非現実的

**ゲイン（得られる価値）:**
- グローバル読者獲得による差別化
- 海外からの新たな収益機会
- 世界中のファンコミュニティ形成

### セカンダリ: 海外のラノベファン

| 属性 | 詳細 |
|------|------|
| 地域 | 北米、韓国、台湾、欧州 |
| 年齢層 | 18-30歳 |
| 志向 | 日本のラノベ・アニメ文化のファン |
| 推定規模 | 約500万人（グローバルラノベ読者） |

**ペイン（課題）:**
- 翻訳版を待つ時間が長い
- 翻訳されない作品が多い
- 翻訳品質にばらつきがある

**ゲイン（得られる価値）:**
- 最新作にすぐアクセスできる
- 母国語で快適に読める
- 作家と直接つながれる

## ペルソナ

### ペルソナ1: 山田太郎（作家志望）

| 項目 | 内容 |
|------|------|
| 年齢・性別 | 28歳、男性 |
| 職業 | ITエンジニア（会社員） |
| 居住地 | 東京都 |
| 執筆歴 | 3年 |
| 現在の活動 | 小説家になろうで異世界ファンタジーを連載中 |

**執筆状況:**
- 週2-3話ペースで更新
- フォロワー数: 約2,000人
- 書籍化経験: なし（狙っている）

**課題:**
- 国内ランキングで埋もれがち
- 差別化が難しい（似た作品が多い）
- 海外読者に届ける手段がない

**Star Novel への期待:**
- 海外読者を獲得して差別化したい
- グローバルでの評価を国内出版社にアピールしたい
- 翻訳なしで世界に発信できるのは魅力的

**利用シナリオ:**
1. 既存作品を Star Novel にも投稿
2. 海外読者からのコメント・反応を確認
3. 人気が出れば Star Novel をメインプラットフォームに

### ペルソナ2: Emily Chen（海外読者）

| 項目 | 内容 |
|------|------|
| 年齢・性別 | 24歳、女性 |
| 職業 | 大学院生 |
| 居住地 | アメリカ・カリフォルニア州 |
| 言語 | 英語（母国語）、日本語（N3レベル） |

**読書習慣:**
- 週に3-5時間ラノベを読む
- 好きなジャンル: 異世界転生、恋愛ファンタジー
- 利用プラットフォーム: Novel Updates、公式翻訳版

**課題:**
- 好きな作品の翻訳が追いつかない
- 翻訳されない作品が多い
- 機械翻訳は品質が悪くて読みづらい

**Star Novel への期待:**
- 日本の最新作をすぐに英語で読みたい
- AI翻訳でも読みやすければ問題ない
- 好きな作家を直接応援したい

**利用シナリオ:**
1. Twitter で Star Novel の存在を知る
2. 気になる日本の作品を英語で試し読み
3. 気に入った作家をフォロー、投げ銭で応援

## 獲得戦略

### 3フェーズ獲得モデル

```
Phase 1: クローズドβ
├── 招待制で厳選した作家 50-100名
├── インフルエンサー作家を優先
└── 永久プレミアム特典で動機付け

Phase 2: オープンβ / 正式ローンチ
├── 作家オーガニック獲得（SNS、口コミ）
├── 海外読者獲得（Reddit、SEO）
└── コミュニティ形成開始

Phase 3: グロース
├── 有料広告による読者獲得
├── パートナーシップ（出版社、アニメ制作）
└── 追加言語展開で新規市場開拓
```

## ユーザーストーリー

### 作家向け

| ストーリー | 優先度 |
|-----------|--------|
| 作家として、日本語で小説を投稿したい。世界中の読者に読んでもらうために。 | Must |
| 作家として、用語集を登録したい。固有名詞が正しく翻訳されるために。 | Must |
| 作家として、海外読者からのコメントを読みたい。フィードバックを得るために。 | Should |
| 作家として、読者の属性を分析したい。どの国で人気があるか知るために。 | Should |
| 作家として、投げ銭を受け取りたい。創作活動を続けるために。 | Want |

### 読者向け

| ストーリー | 優先度 |
|-----------|--------|
| 読者として、日本のラノベを母国語で読みたい。言語の壁なく楽しむために。 | Must |
| 読者として、お気に入りの作品をブックマークしたい。続きを追いかけるために。 | Must |
| 読者として、作家にコメントしたい。感想を伝えるために。 | Should |
| 読者として、作家に投げ銭したい。応援の気持ちを形にするために。 | Want |
| 読者として、オフラインで読みたい。通勤中に読むために。 | Want |
