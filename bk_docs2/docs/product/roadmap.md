## 概要

Star Novel は段階的にリリースし、各フェーズでPMF（Product-Market Fit）を検証しながら成長します。

---

## フェーズ一覧

| Phase | 名称 | 目標 | マネタイズ |
|-------|------|------|-----------|
| 0 | Foundation | 基盤構築・設計 | なし |
| 1 | Closed Beta | 初期フィードバック収集 | なし |
| 2 | Open Beta | 初期ユーザー獲得・PMF検証 | 広告のみ |
| 3 | Growth | ユーザー基盤拡大・収益化開始 | 広告 + サブスク |
| 4 | Scale | グローバル展開・本格収益化 | フルマネタイズ |

## Phase 1: Closed Beta

**目標:** 招待作家からのフィードバック収集、プロダクト改善

### 概要

| 項目 | 内容 |
|------|------|
| 期間 | 2-4週間 |
| 参加者 | 招待制、50-100名の作家 |
| 選定基準 | なろう/カクヨムで連載中、SNSフォロワー1000人以上 |
| インセンティブ | 永久プレミアム会員、正式ローンチ時の優先露出 |

### 機能スコープ

| 機能 | 含む |
|------|------|
| 作家機能 | アカウント作成、作品投稿、エディタ |
| 翻訳機能 | 日本語→英語の自動翻訳 |
| 読者機能 | 閲覧、ブックマーク |
| 分析 | 基本的なアクセス統計 |

### KPI

| 指標 | 目標 |
|------|------|
| β参加作家数 | 50-100人 |
| フィードバック回収率 | 80% |
| 重大バグ件数 | 0 |
| NPS | 30以上 |

## Phase 2: Open Beta / 正式ローンチ

**目標:** 初期ユーザー獲得、PMF検証

### 機能スコープ

| 機能 | 含む |
|------|------|
| 作家機能 | 用語集管理、基本統計 |
| 翻訳機能 | 日本語→英語（広告付き） |

| 指標 | 目標 |

| 登録作家数 | 500人 |
| 投稿作品数 | 200作品 |
| MAU（読者） | 5,000人 |
| 翻訳利用率 | 80% |
| 読者リテンション（7日） | 20% |
| オーガニック流入比率 | 60% |

### ローンチキャンペーン

#### 「世界デビューキャンペーン」

| 項目 | 内容 |
|------|------|
| 期間 | ローンチ後1ヶ月 |
| 対象 | 期間中に作品を投稿した作家 |
| 特典 | 全言語翻訳無料（通常は英語のみ無料） |
| 目的 | 初期コンテンツの充実 |

## 収益モデル

### 収益構造

```
┌─────────────────────────────────────────────────┐
│              基本層（無料）                       │
│  - 日本語での執筆・公開                          │
│  - 英語への自動翻訳（広告付き）                   │
│  - 基本的な読者機能（閲覧、ブックマーク）          │
├─────────────────────────────────────────────────┤
│            作家向けプレミアム                     │
│  - 追加言語への翻訳（韓国語、中国語など）          │
│  - 詳細アナリティクス（読者属性、離脱率など）      │
│  - 用語集の高度な管理機能                        │
│  - 優先サポート                                  │
├─────────────────────────────────────────────────┤
│            読者向けプレミアム                     │
│  - 広告非表示                                    │
│  - 先読み機能（作家が設定可能）                   │
│  - オフライン閲覧                                │
│  - 高品質翻訳オプション（将来）                   │
├─────────────────────────────────────────────────┤
│              投げ銭・応援                        │
│  - 作家への直接支援                              │
│  - プラットフォーム手数料: 15-20%                │
└─────────────────────────────────────────────────┘
```

## リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| 初期コンテンツ不足 | 読者が来ない | β作家への優遇、自社コンテンツ投入 |
| 翻訳品質への不満 | 離脱 | フィードバック機能、継続改善 |
| 競合の追随 | シェア低下 | 先行者優位、コミュニティ強化 |
| 作家の流出 | コンテンツ減少 | 独自機能、収益化支援 |
| 翻訳APIコストの増大 | 利益圧迫 | 使用量ベースの課金、キャッシュ戦略 |
