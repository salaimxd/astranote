## 統合的バリュー・プロポジション

> NFT Book Marketを使うことで、**著者は作品の真の価値を収益化し、読者はサービス終了後も永続的に読める真の所有権を得られる、公正で透明な次世代出版体験**を実現します。
>
> 無限供給モデルにより投機を排除し、健全な読書文化を守りながら、Arweave保存とオープンソースリーダーで200年先まで読書を保証します。

## ペルソナ1: インディーズ著者 - 佐藤ユキ

### 主要な課題

- **課題1: プラットフォームへの高額手数料** - Amazon KDPで売上の30-35%を徴収され、収益性が低い
- **課題2: 読者との関係構築が困難** - 購入者データが取得できず、ファンコミュニティを作れない
- **課題3: デジタル作品の価値が低く見られる** - 電子書籍は紙より安く設定しないと売れず、コレクション性もゼロ

### 提供される価値

- **価値1: 業界最高水準の収益還元** - 初回販売85%、二次流通10%のロイヤリティで持続可能な創作活動を実現
- **価値2: 読者との直接的なつながり** - NFT所有者限定のDiscordコミュニティ、AMAイベント、限定コンテンツで強固なファンベース構築
- **価値3: 作品の資産価値向上** - 限定版NFT、サイン入りエディション、初版特典により「コレクタブル」としての付加価値

### ビジネスインパクト

- **収益増加**: Amazon KDPと比較して、同じ販売部数でも収益が **+50〜70%向上**
- **コミュニティ形成**: 購入者限定コミュニティで **NPS(Net Promoter Score)が60以上** を達成、口コミによる新規読者獲得
- **長期収益**: 二次流通ロイヤリティにより、発行から **1年後も継続収益** を得られる

### 具体的なユースケース

**ケース: 新作小説のリリース**

1. **従来(Amazon KDP)**:
   - 価格: ¥1,000 → 手数料35%引かれて著者受取 ¥650
   - 100部販売で ¥65,000 の収益(1回限り)
   - 読者との接点なし、購入者リスト取得不可

2. **NFT Book Market**:
   - 価格: ¥1,200(通常版) + ¥3,000(初版限定100部)
   - 手数料15%で著者受取 ¥1,020(通常版) / ¥2,550(限定版)
   - 通常版80部 + 限定版20部販売で **¥132,600 の収益** (+104%)
   - 限定版購入者は著者Discord参加権 → 次回作の先行リリース告知で **リピート購入率65%**
   - 6ヶ月後、限定版の二次流通3件発生 → ロイヤリティ **追加 ¥2,700**

## ペルソナ2: NFTコレクター読者 - 田中ケン

### 主要な課題

- **課題1: 電子書籍に所有感がない** - Kindleは実質「ライセンス」で、アカウント停止リスクがあり資産性ゼロ
- **課題2: 好きな著者を直接支援できない** - Patreon等は別扱いで、書籍購入と統合されていない
- **課題3: NFTアートの実用性不足** - 所有しているNFTは「見るだけ」で、実際に使う機会がない

### 提供される価値

- **価値1: 真のデジタル所有権** - ブロックチェーンで保証された所有権により、転売・貸出・贈与が自由。プラットフォーム依存なし
- **価値2: 読書+コレクションの融合** - NFTの所有体験と、実用的な「読む」体験の両立。物理本のように「本棚に並べる喜び」をデジタルで再現
- **価値3: 著者コミュニティへのアクセス** - 限定版NFT保有者は著者との直接交流イベント、未公開コンテンツへのアクセス権

### ビジネスインパクト(読者視点)

- **真の所有体験**: Kindleと異なり、サービス終了後も永続的に読める安心感。Arweave保存で200年保証
- **コミュニティ参加**: NFT保有者限定イベントで **著者と直接交流**、次回作のストーリーアイデアに参加できる特別体験
- **コレクション管理**: 所有するNFT本はウォレットで一元管理、デジタル本棚として可視化
- **譲渡の自由**: 読み終わった本を友人に譲渡・贈与可能。物理本と同じ文化をデジタルで実現

### 具体的なユースケース

**ケース: お気に入り著者の新刊発売**

1. **従来(Kindle)**:
   - 価格: ¥1,200で購入
   - 読み終わったら本棚の肥やし、再読率低い
   - 著者との接点なし、「応援している」という実感が薄い

2. **NFT Book Market**:
   - 価格: ¥1,200で通常版NFTを購入(無限供給、いつでも定価で購入可能)
   - NFT保有者限定特典:
     - 著者の執筆裏話音声コメンタリー付き(別途¥500で購入可能な特典NFT)
     - 月1回の著者AMAイベント参加権(Discordライブ)
     - 未公開短編小説アクセス権(NFT保有者のみ)
   - **サービス終了後も読める安心感**: Arweaveに永続保存、オープンソースリーダーで読書継続可能
   - 読み終わったら友人に譲渡可能(二次流通価格も定価¥1,200が基準)
   - 「初版購入者」バッジ(無料NFT)を取得、コミュニティ内で著者への応援を可視化

## ケーススタディ

### 事例1: インディーズSF作家・山田タロウ氏(仮名)

- **課題**: Amazon KDPで5年間活動するも、月収5万円前後で伸び悩み。読者からの反応が見えず、モチベーション低下。
- **解決策**: NFT Book Marketで新作を¥1,500で販売(無限供給)。全購入者にDiscordコミュニティ招待、月1回のAMA参加権を提供。特典として「著者音声コメンタリーNFT」(¥300)と「初版購入者バッジNFT」(無料)を発行。
- **成果**:
  - 初月で通常版150部 + 音声コメンタリーNFT80部を販売 → **収益¥245,100**(従来比 +190%)
  - Discord参加者95名のアクティブコミュニティ形成 → 次回作の先行予約が **45部確約**
  - 6ヶ月後、Arweave永久保存オプション(+¥2)を追加 → 「200年保証」がマーケティングポイントに
  - 「読者の顔が見えるようになり、執筆のモチベーションが劇的に向上した」「サービス終了リスクがないので、長期的な読者との関係構築に集中できる」

- **引用**: 「KDPでは売上の数字しか見えませんでした。でもNFT Book Marketでは、読者が本当に作品を愛してくれていることを実感できます。Discordで『次はこんなキャラが読みたい!』と言ってもらえると、『この人たちのために書きたい』と思える。それが何より大きいです」 - 山田タロウ氏(SF作家)

### 事例2: ビジネス書読者・鈴木ハナコ氏(仮名/ NFTコレクター)

- **課題**: Kindle Unlimitedで月20冊読むが、「所有している」実感がなく虚しさを感じていた。好きな著者を応援したいが、Kindleでは「ただ読むだけ」。また、Amazonがサービス終了したら全ての本が読めなくなる不安。
- **解決策**: NFT Book Marketで、好きなマーケティング専門家の新刊を¥1,800で購入(無限供給)。全購入者特典として、著者の週次ニュースレター、四半期ライブセミナー参加権を取得。初版購入者バッジNFT(無料)を獲得。
- **成果**:
  - NFT本を3ヶ月で8冊購入 → **デジタル本棚が「自慢のコレクション」に**
  - **Arweave永久保存により、サービス終了後も読める安心感**: 「Kindleで積読している100冊が、Amazonの都合で消える不安から解放された」
  - 読み終わった1冊を友人に譲渡 → 友人もNFT Book Marketを知り、著者コミュニティに参加
  - 著者とのライブセミナーで直接質問でき、**「ファン」から「コミュニティメンバー」に**

- **引用**: 「Kindleで読んでいた時は『消費』でしたが、NFT Book Marketでは『所有』と『応援』の感覚です。一番大きいのは、サービス終了後も読めるという安心感。Kindleの100冊が、Amazonの都合で消えるかもしれない不安から解放されました。デジタル本棚が『本当に自分のもの』になった実感があります。それに、読み終わった本を友人に譲れるのも、物理本のような感覚で気に入っています」 - 鈴木ハナコ氏(読者・NFTコレクター)
